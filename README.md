dmp2player
==========

Dreambox P2P player based on Tribler sources (http://www.tribler.org/).


Installation
============


 *  Python >= 2.5
 *  M2Crypto >= 0.16
 *  APSW aka. python-apsw >= 3.6.x (Python wrappers for SQLite database)

to install on Ubuntu:

sudo apt-get install python-m2crypto python-apsw
